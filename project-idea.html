Project Idea

 

Overview:

 

The concept for this project is a tool that easily and securely combines all of an individualâ€™s communication tools, methods and applications or programs that they use into one platform. This would enable the user to access their social media platforms (Facebook, Twitter, Snapchat, Instagram, LinkedIn etc), email services (Gmail, iCloud Mail, Yahoo Mail, Microsoft etc), messaging services (Facebook messenger, iMessage, Text, Discord, MS teams etc) to be accessed via one program or application. This would be available for mobile devices, computers and tablets across all operating systems.

 

Motivation:

 

The reason for this project is purely around ease of access. Currently, users are required to run a unique application, program or web browser to access each of these individual services with unique user-names and passwords to be used for each. By combining all the above mentioned services into 1 application or program, the user will not be required to run a different program, application or web browser each time they wish to access these. Not only this, but once this program/application was created, the user would be able to set 1 master login credential that would enable access to all of these services.

 

Description:

 

The idea is to create a program or application that will centralize every social media platform, messenger service, email service and any other communication tool that a user or business may need. By enabling a fully centralized program or application to have all of these services inbuilt, this will enable the user/business to be able to access, monitor and control all of these inside 1 program.

 

There are several issues that may be encountered if this project were to go from the drawing board stage to development; these include but are not limited to copyright infringement, security, marketing, brand awareness, Hardware limitations and a lack of willingness in the uptake of this service from the end user, the service provider/s and/or owners of these communication mediums.

 

Even if the above mentioned issues were not something that was going to be encountered, building this type of program or application would have an extensive development time as it would need to be designed for use across many different operating systems each with their own inherent flaws and vulnerabilities. The program/application would need to be built in Linux/Unix programming language for Android and iOS devices then Dos for any windows based devices.

 

Cyber security is going to be one of the major issues that development of this project is going to face. In this day an age where hackers are becoming more and more efficient at social engineering and exploiting weaknesses within IT infrastructure, the development team will have to consult with the IT cyber security teams within each of the applicable service providers companies to find the safest way in which to on-board the service in a safe, secure and user friendly way. Also, the development team will have to work hand in hand with the companies that this platform is going to be used on (Microsoft, Apple, Samsung etc) to eliminate any potential ways to access the platform from device exploits or back doors.

 

One of the interesting things about attempting a project of this scale is the amount of variation in requirements and knowledge required to make this. From the different programming languages required to write it, incorporating many different services into it, generating brand awareness, the different hardware services required to run it ie servers and cloud networking, to the cyber security protocols that would have to be built in and monitored and updated on an increasingly frequent scale, the sheer size or scope of a project like this would incorporate so many different types of service, hardware, legal approval and businesses that anyone interested in an ongoing, long term IT profession would be able to have access to almost all IT professions and aspects within the one role.

 

Tools and Technologies:

 

The tools, technologies and services required for this project are many and varied. Programmers, mobile app developers, website developers, cloud networking technicians, cyber security experts, project managers, marketing developers to social media managers, this project requires IT staff and business staff from all aspects of life. Linux/Unix as well as iOS development suits will be required, as well as Dos and C++. Java and ruby/python will also be required.

 

Skills Required:

 

Competency in iOS, Android, Linux/Unix and Windows development suits is a must. Java, C++, ruby/python, Powershell, terminal/command prompt, Website development and business modeling is a must. Other communication tools/programs as well as a competent understanding of the legal framework, copyright legislation and business modeling is also a must.

 

Outcome:

 

Upon successful completion of this project, if all desired outcomes are achieved, the result would be a secure, all in 1 communication hub/platform that enables users from all walks of life to be able to manage and access any/all services/programs that they use for any and all forms of communication and social media. Being able to access and launch any of these services from 1 central, secure location should improve ease of access for the end user as well as enable them to manage multiple different services easily.
